# react-native-okhi

The OkHi React Native library enables you to collect and verify addresses from your users

## Installation

```sh
npm install react-native-okhi
```

## Usage

```js
import { multiply } from "react-native-okhi";

// ...

const result = await multiply(3, 7);
```
